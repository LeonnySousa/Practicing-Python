Este repositório possui 3 projetos:

1 - BancoPy: Código que simula o funcionamento de um banco ao cadastrar clientes, realizar depósitos, saques e transferências.

2 - Calculator_GAME: Código que desafia o usuário a responder calculos em diferentes níveis de dificuldade.

3 - PyMarket: Código que simula o funcionamento de um mercado ao cadastrar produtos e fechar pedido de compras.
